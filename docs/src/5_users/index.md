# Benutzerverwaltung

Um das Erstellen und Verwalten von Übungen zu erleichtern, besitzt die FüSim Digital eine
Benutzerverwaltung. Nutzer können sich auf der Website ein Benutzerkonto erstellen und dort Übungen
sowie Übungsvorlagen erstellen.

## Registrierung

Um ein neues Benutzerkonto zu erstellen, kann mit dem Button [<kbd>Registrieren</kbd>](https://fuesim-manv.de/api/auth/register) oben rechts auf der Startseite das [Registrierungsformular](https://fuesim-manv.de/api/auth/register) geöffnet werden. Nach Angabe von einer E-Mail-Adresse, Vor- und Nachnamen sowie einem Passwort müssen noch unsere [Nutzungsbedingungen](https://fuesim-manv.de/about/terms) akzeptiert werden. Bevor man sich das erste Mal anmelden kann, muss zudem die E-Mail-Adresse verifiziert werden: Dazu muss der Benutzer auf den Link in der Bestätigungs-E-Mail klicken. Die Daten werden gemäß unserer [Datenschutzerklärung](https://fuesim-manv.de/about/privacy) verarbeitet.

## Anmeldung

Nach der Registrierung kann man sich mit dem Benutzerkonto auf der Startseite über [<kbd>Anmelden</kbd>](https://fuesim-manv.de/api/auth/oidc-redirect) jederzeit anmelden, um Zugriff auf seine [Übungen und Übungsvorlagen](../4_editing/) zu erhalten. Es tauchen dann oben in der Mitte zwei zusätzliche Menüpunkte auf, wo die entsprechenden Objekte aufgelistet werden.

## Benutzerkonto verwalten

Um eine 2-Faktor-Authentifizierung einzurichten (stark empfohlen), das Passwort zu ändern oder andere persönliche Daten zu aktualisieren, kann über den eigenen Namen oben rechts und dann den Menüpunkt [<kbd>Konto verwalten</kbd>](https://fuesim-manv.de/api/auth/self-service) die Kontoverwaltung geöffnet werden. Auf der Seite [<kbd>Persönliche Angaben</kbd>](https://auth.fuesim-manv.de/realms/fuesim-digital/account/) kann dann der eigene Name aktualisiert werden und über [<kbd>Kontosicherheit</kbd>](https://auth.fuesim-manv.de/realms/fuesim-digital/account/account-security/signing-in) ein zweiter Faktor hinzugefügt oder das Passwort geändert werden. Die Benutzerverwaltung führt dabei Schritt für Schritt durch die notwendigen Aktionen.

## Benutzerkonto löschen

Zum Löschen eines Benutzerkontos kann über [E-Mail](mailto:kontakt@manv-simulation.de) Kontakt zu uns aufgenommen werden.
