# Simulation

FüSim Digital besitzt Simulationsfunktionen, die die Aufgaben von Übungsteilnehmenden übernehmen können, falls es für ein Szenario nicht genügend Teilnehmende gibt. Die Simulation besitzt einen großen Umfang von Einstellungsmöglichkeiten. Sie sollten sich daher zuerst mittels dieser Dokumentation mit den Funktionen vertraut machen, bevor Sie sie in einer Übung einsetzen. Korrigierende Eingriffe in eine laufende Simulation sind nur beschränkt möglich.

- Im Abschnitt [Allgemeines](1_general.md) finden Sie einen Überblick über die Konzepte der Simulation, ihre Funktionen und Anwendungsfälle.
- Im Abschnitt [Verhaltensweisen](2_behaviors.md) werden die einzelnen Funktionen, die die Simulation bietet, im Detail erläutert.
- Im Abschnitt [Schnittstellenfunker](3_ifs.md) ist die Bedienung der Benutzeroberfläche für Schnittstellenfunker erläutert. Eine Beschreibung dieser Rolle finden Sie [im Abschnitt Allgemeines](1_general.md#funkspruch--schnittstellenfunker).
