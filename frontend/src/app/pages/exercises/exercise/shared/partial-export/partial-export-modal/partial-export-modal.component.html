Partieller Export

Leider steht der partielle Export aufgrund von Umbauarbeiten für eine neue Version der Software, welche umfangreiche Funktionen zum Verwalten von Vorlagen enthalten wird, nicht mehr zur Verfügung. Bitte benutzen Sie alternativ die Möglichkeit, einen kompletten Übungsexport durchzuführen und nach dem Import nicht benötigte Elemente zu löschen. Der Import von alten partiellen Exporten ist weiterhin unter Vorlagen importieren möglich.
